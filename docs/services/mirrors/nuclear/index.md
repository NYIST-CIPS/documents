# mirrors2

H3C NaviData 5200 G2 承担了目前 mirror 的几乎所有流量(通过 kruby 反向代理)。

| 参数  |                配置                |
| :---: | :--------------------------------: |
|  CPU  |          双路 E5-2630 v4           |
| 内存  |            384 GB DDR4             |
| 存储  | 16 TB \* 10 (HDD), 500 GB \*2 (SSD), 4TB \1 (nvme) |
| 网络  |            10 Gbps \* 1             |
